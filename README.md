# CubeFS (cubefs)
CubeFS is a CNCF graduated cloud-native distributed file system supporting POSIX, HDFS, and S3-compatible object storage protocols. It provides multi-tenancy, multi-AZ deployment, and erasure coding for large-scale storage.

## APIs

### CubeFS S3-Compatible API
CubeFS provides an S3-compatible object storage interface through its ObjectNode component, supporting standard S3 operations including bucket management, object CRUD, multipart uploads, and access control.

**Human URL:** [https://cubefs.io/docs/master/user-guide/objectnode.html](https://cubefs.io/docs/master/user-guide/objectnode.html)

### CubeFS Master API
The CubeFS Master API provides HTTP endpoints for cluster management including volume creation, data partition management, node management, and cluster status monitoring.
